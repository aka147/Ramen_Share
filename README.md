# ラーメンShare
ラーメン投稿サイト

## サイト概要
自分の好みのラーメン、ラーメン屋を投稿できるサイト


### サイトテーマ
好きなラーメン、ラーメン屋をみんなでシェアしよう！！

### テーマを選んだ理由
ラーメン好きがラーメンを投稿することにより、好きなラーメン屋を知ってもらうためです。
その結果、そのラーメン屋さんがより盛り上がることができれば嬉しいと考えて今回の作成に至りました。

### ターゲットユーザ
- ラーメンを好きな人
- 好きなラーメンをシェアしたい人
- ラーメン屋

### 主な利用シーン
- お気に入りのラーメン、ラーメン屋についての情報を投稿し共有する際に活用。
- 他の投稿者のラーメン情報を閲覧したい際に活用。

## 開発環境
- HTML&CSS
- Ruby
- Javascript
- フレームワーク
- Ruby on rails (5.2.4)
